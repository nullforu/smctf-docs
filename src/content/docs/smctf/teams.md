팀은 하나의 [부문](/smctf/divisions)에 속하면서 문제 풀이에 참여하는 단위입니다. 
팀은 하나 이상의 [유저](/smctf/users)로 구성되며, 팀 내에서 풀이한 문제는 공유되어 다른 팀원이 이미 풀이된 문제를 다시 풀 필요가 없습니다. 

때문에 부문 내에서 [스코어링 시스템](/smctf/advanced/dynamic-score)도 팀 단위로 운영되며, 부문 내에서 서로 경쟁하는 단위가 됩니다. 
부문과 마찬가지로 유저/팀/부문 간 매칭을 사전에 다시 점검하세요.
