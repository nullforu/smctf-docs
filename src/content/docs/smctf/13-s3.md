SMCTF는 문제 풀이에 필요한 소스코드나 바이너리 파일 등을 ZIP 파일 형태로 압축하여 AWS S3 버킷에 업로드하고 이를 다운로드 할 수 있습니다. 이때 AWS S3 Presigned URL을 사용합니다.

업로드는 문제를 생성하거나 수정할 때 ZIP 파일을 선택하여 업로드할 수 있습니다. 업로드된 ZIP 파일은 UUID로 변경되어 AWS S3 버킷에 저장됩니다.

![image](images/13-s3/image.png)

사용자는 문제에 대한 상세 정보 모달에서 해당 ZIP 파일을 다운로드할 수 있습니다. 다운로드 링크느 일정 시간이 지나면 자동으로 만료됩니다.

![image](images/13-s3/image-1.png)

아래와 같이 UUID로 저장됩니다. (다운로드 시 자동으로 원본 이름으로 변경되어 다운로드됩니다.)

![image](images/13-s3/image-2.png)

Forensics, Network 등의 카테고리에서 GB 단위의 큰 파일(예: `.pcap`, `.vmdk` 등)을 제공해야 하는 경우 인프라 팀에 별도로 문의하세요. Presigned URL을 통해 대용량의 파일을 업로드하는데 있어 제약이 있을 수 있습니다.
