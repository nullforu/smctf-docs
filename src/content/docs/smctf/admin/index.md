관리자 페이지는 RBAC에 따라 Admin 역할을 가진 유저만 접근할 수 있으며, 초기 Admin 역할 할당 방법은 [설치](/smctf/20-install-and-docker) 문서에서 확인할 수 있습니다.

관리자 페이지는 크게 아래와 같은 탭으로 구성되어 있습니다.

- Create Challenge
- Challenge Management
- Users (User Management)
- Teams (Team Management)
- Stacks (Stack Management)
- Registration Keys
- Site Configuration
- Report

각각의 탭에 대한 자세한 설명은 다음 페이지에서 설명합니다. 각 탭은 왼쪽 사이드바에서 선택할 수 있으며 모바일에선 선택 메뉴에서 선택할 수 있습니다.
