기존의 문제를 수정하거나 삭제할 수 있습니다. 각 필드는 개별적으로 수정할 수 있으며, 일부 필드는 비어있지 않도록 요구할 수 있습니다.

![image](images/2-chall-manage/image.png)

Action 중 Edit을 클릭하면 문제에 대한 정보를 수정할 수 있으며, Delete를 클릭하면 문제를 삭제할 수 있습니다. 

import { Aside } from '@astrojs/starlight/components';

<Aside type="danger">
문제를 삭제할 경우 해당 문제에 대한 모든 플래그 제출 기록과 점수 등이 모두 삭제됩니다.
</Aside>

![image](images/2-chall-manage/image-1.png)

기존의 플래그 값은 로드되지 않습니다. 값을 수정하면 새로운 플래그 값으로 업데이트됩니다. 

![image](images/2-chall-manage/image-2.png)
