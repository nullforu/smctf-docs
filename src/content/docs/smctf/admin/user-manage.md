[유저](/smctf/users)에 대해 팀 이동 및 차단(제한) 등을 관리할 수 있는 탭입니다. **팀 이동시 해당 유저가 풀이한 기록이 전부 새로운 팀으로 이전됩니다. 주의하여 사용하세요.**

![User Manage 1](/smctf/imgs/admin_users_1.png)

_우측의 Block User 버튼을 클릭하여 해당 유저를 차단할 수 있습니다. 이에 대한 사유가 필수적으로 입력되어야 합니다._

![User Manage 2](/smctf/imgs/admin_users_2.png)

_차단된 유저는 언제든 다시 Unblock 버튼을 클릭하여 차단을 해제할 수 있습니다._

![User Manage Ban](/smctf/imgs/admin_users_ban.png)
